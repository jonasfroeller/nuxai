# Nuxai

Some chat app.

## Built with

[Nuxt 3](https://nuxt.com/docs/getting-started/introduction), [Nitro](https://nitro.unjs.io), [Vite](https://vitejs.dev) and [Vue 3](https://vuejs.org).

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm run dev
```

## Production

```bash
pnpm run build
```

Locally preview production build:

```bash
pnpm run preview
```

## Deployment

```bash
vercel login
```

Deploy to vercel:

```bash
vercel
```

Connect git repository:

```bash
vercel git connect
```
